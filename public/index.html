วัตถุประสงค์: สร้างตารางแสดงคุณลักษณะส่วนบุคคลของกลุ่มตัวอย่าง สำหรับบทที่ 4

ผลลัพธ์: ตารางความถี่ ร้อยละ พร้อมระบุ N รวม — พร้อมใช้ในวิทยานิพนธ์

ตัวอย่างตัวแปร: เพศ, อายุ, ระดับการศึกษา, สถานภาพ, รายได้, อาชีพ, ประสบการณ์

รองรับ:

  • ตัวแปรเชิงกลุ่ม (Categorical) — นับความถี่ ร้อยละ เช่น เพศ ระดับการศึกษา
  • ตัวแปรเชิงปริมาณ (Numeric) — แสดง Mean, S.D., Min, Max เช่น อายุ รายได้

เลือกตัวแปรคุณลักษณะส่วนบุคคล

เลือกตัวแปรที่ต้องการนำมาสร้างตารางข้อมูลทั่วไป เช่น เพศ อายุ ระดับการศึกษา สถานภาพสมรส รายได้

⚙️ ตั้งค่าช่วงตัวแปรเชิงปริมาณ

📝 Descriptive Analysis

📖 เมื่อไหร่ควรใช้ Descriptive Analysis ?

วัตถุประสงค์: สรุปลักษณะข้อมูลเบื้องต้น เช่น ค่าเฉลี่ย ส่วนเบี่ยงเบนมาตรฐาน

เงื่อนไข:

  • ใช้ได้กับข้อมูลเชิงปริมาณ (ตัวเลข)

ข้อมูลที่ต้องมี: ตัวแปรเชิงปริมาณอย่างน้อย 1 ตัว

ตัวอย่าง: สรุปคะแนนสอบของนักเรียน 30 คน

🔢 Numeric Variable Analysis

📖 เมื่อไหร่ควรใช้ Numeric Variable Analysis ?

วัตถุประสงค์: วิเคราะห์ตัวแปรเชิงปริมาณอย่างละเอียด

เงื่อนไข:

  • ตัวแปร Numeric

ข้อมูลที่ต้องมี: ตัวแปรเชิงปริมาณ

ตัวอย่าง: วิเคราะห์อายุ รายได้ คะแนน

🏷️ Nominal Variable Analysis

📖 เมื่อไหร่ควรใช้ Nominal Variable Analysis ?

วัตถุประสงค์: วิเคราะห์ตัวแปรเชิงกลุ่ม (ความถี่ ร้อยละ)

เงื่อนไข:

  • ตัวแปร Categorical

ข้อมูลที่ต้องมี: ตัวแปรเชิงกลุ่ม

ตัวอย่าง: เพศ ระดับการศึกษา สถานภาพ

📏 Likert Scale Analysis

📖 เมื่อไหร่ควรใช้ Likert Scale Analysis ?

วัตถุประสงค์: วิเคราะห์แบบสอบถามมาตราส่วนประมาณค่า

เงื่อนไข:

  • ข้อมูลเป็นมาตราส่วน (3 หรือ 5 ระดับ)

ข้อมูลที่ต้องมี: ข้อคำถามแบบ Likert Scale

ตัวอย่าง: ความพึงพอใจต่อบริการ

⚙️ ตั้งค่าเกณฑ์การแปลผล (Interpretation Criteria)

กำหนดช่วงค่าเฉลี่ยและความหมายสำหรับแต่ละระดับ

📐 Interval Analysis

📖 เมื่อไหร่ควรใช้ Interval Analysis ?

วัตถุประสงค์: จัดกลุ่มข้อมูลตามช่วง (Class Interval)

เงื่อนไข:

  • ตัวแปร Numeric

ข้อมูลที่ต้องมี: ตัวแปรเชิงปริมาณ

ตัวอย่าง: จัดกลุ่มอายุ 18-25, 26-35

⚙️ ตั้งค่าช่วง (Interval) แต่ละตัวแปร

กรุณาเลือกตัวแปรก่อน แล้วกดปุ่มตั้งค่า

🔍 Outlier Detection

📖 เมื่อไหร่ควรใช้ Outlier Detection ?

วัตถุประสงค์: ตรวจหาค่าผิดปกติ (Outlier) ด้วยวิธี IQR

เงื่อนไข:

  • ตัวแปร Numeric

ข้อมูลที่ต้องมี: ตัวแปรเชิงปริมาณ

ตัวอย่าง: ตรวจสอบคะแนนสอบที่มีค่าผิดปกติ

📉 Normality Test

📖 เมื่อไหร่ควรใช้ Normality Test ?

วัตถุประสงค์: ทดสอบว่าข้อมูลมีการแจกแจงปกติหรือไม่

เงื่อนไข:

  • ใช้กับข้อมูลเชิงปริมาณ
  • ตัวอย่างมากกว่า 3 ค่า

ข้อมูลที่ต้องมี: ตัวแปรเชิงปริมาณ

ตัวอย่าง: ตรวจสอบก่อนเลือกใช้สถิติ Parametric หรือ Non-Parametric

📊 Independent Samples t-test

📖 เมื่อไหร่ควรใช้ Independent t-test ?

วัตถุประสงค์: เปรียบเทียบค่าเฉลี่ยของ 2 กลุ่มที่เป็นอิสระจากกัน

เงื่อนไข:

  • ข้อมูลแจกแจงปกติ
  • ความแปรปรวนเท่ากัน (หรือใช้ Welch's)
  • 2 กลุ่มอิสระ

ข้อมูลที่ต้องมี: ตัวแปรตาม (DV) เชิงปริมาณ 1 ตัว + ตัวแปรอิสระ (IV) เชิงกลุ่ม 2 กลุ่ม

ตัวอย่าง: เปรียบเทียบคะแนนสอบระหว่างกลุ่มทดลองกับกลุ่มควบคุม

📊 Paired Samples t-test

📖 เมื่อไหร่ควรใช้ Paired t-test ?

วัตถุประสงค์: เปรียบเทียบค่าเฉลี่ยก่อน-หลัง ในกลุ่มเดียวกัน

เงื่อนไข:

  • ข้อมูลผลต่างแจกแจงปกติ
  • เป็นข้อมูลคู่จากกลุ่มเดียวกัน

ข้อมูลที่ต้องมี: ตัวแปรก่อน (Before) + ตัวแปรหลัง (After) เชิงปริมาณ

ตัวอย่าง: เปรียบเทียบคะแนนก่อน-หลังเรียนด้วย CAI

📊 One-way ANOVA

📖 เมื่อไหร่ควรใช้ One-way ANOVA ?

วัตถุประสงค์: เปรียบเทียบค่าเฉลี่ยตั้งแต่ 3 กลุ่มขึ้นไป

เงื่อนไข:

  • ข้อมูลแจกแจงปกติ
  • ความแปรปรวนเท่ากัน
  • กลุ่มอิสระ

ข้อมูลที่ต้องมี: ตัวแปรตาม (DV) เชิงปริมาณ + ตัวแปรอิสระ (IV/Factor) ตั้งแต่ 3 กลุ่ม

ตัวอย่าง: เปรียบเทียบผลสัมฤทธิ์ 3 วิธีสอน

📊 Two-way ANOVA

📖 เมื่อไหร่ควรใช้ Two-way ANOVA ?

วัตถุประสงค์: วิเคราะห์ปฏิสัมพันธ์ (Interaction) ของ 2 ปัจจัย

เงื่อนไข:

  • DV แจกแจงปกติ
  • 2 Factors
  • กลุ่มอิสระ

ข้อมูลที่ต้องมี: ตัวแปรตาม (DV) + 2 ตัวแปรอิสระ (Factors)

ตัวอย่าง: ผลของเพศ x วิธีสอนต่อคะแนน

📊 Repeated Measures ANOVA

📖 เมื่อไหร่ควรใช้ Repeated Measures ANOVA ?

วัตถุประสงค์: เปรียบเทียบค่าเฉลี่ยวัดซ้ำ 3 ครั้งขึ้นไป (Parametric)

เงื่อนไข:

  • ข้อมูลแจกแจงปกติ
  • Sphericity

ข้อมูลที่ต้องมี: ตัวแปรวัดซ้ำ 3 ตัวขึ้นไป

ตัวอย่าง: เปรียบเทียบผลสอบ 3 ครั้ง

📊 ANCOVA

📖 เมื่อไหร่ควรใช้ ANCOVA ?

วัตถุประสงค์: เปรียบเทียบค่าเฉลี่ยโดยควบคุมตัวแปรร่วม

เงื่อนไข:

  • DV แจกแจงปกติ
  • Covariate เป็นตัวเลข

ข้อมูลที่ต้องมี: ตัวแปรตาม (DV) + ตัวแปรอิสระ (IV) + Covariate

ตัวอย่าง: เปรียบเทียบกลุ่มทดลอง-ควบคุม โดยควบคุม Pretest

📐 Mann-Whitney U Test

📖 เมื่อไหร่ควรใช้ Mann-Whitney U ?

วัตถุประสงค์: เปรียบเทียบ 2 กลุ่มอิสระ เมื่อข้อมูลไม่แจกแจงปกติ

เงื่อนไข:

  • ข้อมูลอย่างน้อยเป็นอันดับ (Ordinal)
  • 2 กลุ่มอิสระ

ข้อมูลที่ต้องมี: ตัวแปรตาม + ตัวแปรอิสระ 2 กลุ่ม

ตัวอย่าง: เปรียบเทียบความพึงพอใจ (Likert) ระหว่าง 2 กลุ่ม

หมายเหตุ: ใช้แทน Independent t-test เมื่อข้อมูลไม่ปกติ

📐 Wilcoxon Signed-Rank Test

📖 เมื่อไหร่ควรใช้ Wilcoxon Signed-Rank ?

วัตถุประสงค์: เปรียบเทียบก่อน-หลัง เมื่อข้อมูลไม่แจกแจงปกติ

เงื่อนไข:

  • ข้อมูลเป็นคู่จากกลุ่มเดียวกัน
  • อย่างน้อยเป็นอันดับ

ข้อมูลที่ต้องมี: ตัวแปร Before + After

ตัวอย่าง: เปรียบเทียบทัศนคติก่อน-หลังอบรม (Likert)

หมายเหตุ: ใช้แทน Paired t-test

📐 Kruskal-Wallis H Test

📖 เมื่อไหร่ควรใช้ Kruskal-Wallis H ?

วัตถุประสงค์: เปรียบเทียบตั้งแต่ 3 กลุ่มอิสระ เมื่อข้อมูลไม่ปกติ

เงื่อนไข:

  • กลุ่มอิสระ 3+ กลุ่ม
  • อย่างน้อยเป็นอันดับ

ข้อมูลที่ต้องมี: ตัวแปรตาม + ตัวแปรอิสระ 3+ กลุ่ม

ตัวอย่าง: เปรียบเทียบความพึงพอใจ 3 กลุ่มอายุ

หมายเหตุ: ใช้แทน One-way ANOVA

📐 Friedman Test

📖 เมื่อไหร่ควรใช้ Friedman Test ?

วัตถุประสงค์: เปรียบเทียบข้อมูลวัดซ้ำ 3 ครั้งขึ้นไป แบบ Non-Parametric

เงื่อนไข:

  • ข้อมูลวัดซ้ำจากกลุ่มเดียวกัน 3+ ครั้ง

ข้อมูลที่ต้องมี: ตัวแปรวัดซ้ำตั้งแต่ 3 ตัว

ตัวอย่าง: เปรียบเทียบคะแนนประเมิน 3 รอบ

📐 Chi-Square Test

📖 เมื่อไหร่ควรใช้ Chi-Square Test ?

วัตถุประสงค์: ทดสอบความสัมพันธ์ระหว่างตัวแปรเชิงกลุ่ม 2 ตัว

เงื่อนไข:

  • ตัวแปรเป็นเชิงกลุ่ม (Nominal/Ordinal)
  • Expected Frequency ≥ 5 อย่างน้อย 80%

ข้อมูลที่ต้องมี: ตัวแปรเชิงกลุ่ม 2 ตัว

ตัวอย่าง: ความสัมพันธ์ระหว่างเพศกับความพึงพอใจ

📈 Correlation Analysis

📖 เมื่อไหร่ควรใช้ Correlation Analysis ?

วัตถุประสงค์: วิเคราะห์ความสัมพันธ์ระหว่างตัวแปรเชิงปริมาณ

เงื่อนไข:

  • Pearson = ข้อมูลปกติ
  • Spearman/Kendall = ข้อมูลอันดับหรือไม่ปกติ

ข้อมูลที่ต้องมี: ตัวแปรเชิงปริมาณ 2 ตัวขึ้นไป

ตัวอย่าง: ความสัมพันธ์ระหว่างชั่วโมงอ่านหนังสือกับเกรดเฉลี่ย

📈 Linear Regression

📖 เมื่อไหร่ควรใช้ Linear Regression ?

วัตถุประสงค์: พยากรณ์ตัวแปรตามจากตัวแปรอิสระ

เงื่อนไข:

  • ตัวแปรตามเป็นตัวเลขต่อเนื่อง
  • ความสัมพันธ์เชิงเส้น
  • Residuals ปกติ

ข้อมูลที่ต้องมี: ตัวแปรตาม (DV) 1 ตัว + ตัวแปรอิสระ (IV) 1 ตัวขึ้นไป

ตัวอย่าง: ปัจจัยที่ส่งผลต่อผลสัมฤทธิ์ทางการเรียน

📈 Logistic Regression

📖 เมื่อไหร่ควรใช้ Logistic Regression ?

วัตถุประสงค์: พยากรณ์ตัวแปรตามแบบ 2 ค่า (0/1)

เงื่อนไข:

  • ตัวแปรตามเป็น Binary (0/1)
  • ไม่มี Multicollinearity สูง

ข้อมูลที่ต้องมี: ตัวแปรตาม (DV) แบบ 0/1 + ตัวแปรอิสระ (IV) 1 ตัวขึ้นไป

ตัวอย่าง: ปัจจัยที่ส่งผลต่อการตัดสินใจซื้อสินค้า

🔍 Assumption Tests

📖 เมื่อไหร่ควรใช้ Assumption Tests ?

วัตถุประสงค์: ตรวจสอบข้อตกลงเบื้องต้นของสถิติ

เงื่อนไข:

  • ใช้ก่อนวิเคราะห์สถิติหลัก

ข้อมูลที่ต้องมี: ตัวแปรที่ต้องการตรวจสอบ

ตัวอย่าง: ตรวจ Normality, Homogeneity, VIF

🔧 Reliability Analysis (Cronbach's Alpha)

📖 เมื่อไหร่ควรใช้ Reliability Analysis ?

วัตถุประสงค์: ทดสอบความเชื่อมั่นของเครื่องมือวิจัย (แบบสอบถาม)

เงื่อนไข:

  • ข้อคำถามต้องวัดสิ่งเดียวกัน (Unidimensional)

ข้อมูลที่ต้องมี: ข้อคำถาม (Items) เชิงปริมาณ 2 ข้อขึ้นไป

ตัวอย่าง: ทดสอบความเชื่อมั่นของแบบวัดความพึงพอใจ 5 ข้อ

📏 Effect Size Calculator

📖 เมื่อไหร่ควรใช้ Effect Size Calculator ?

วัตถุประสงค์: วัดขนาดอิทธิพล (Effect Size)

เงื่อนไข:

  • มีผลการทดสอบแล้ว

ข้อมูลที่ต้องมี: ตัวแปรที่ต้องการวัดขนาดอิทธิพล

ตัวอย่าง: Cohen's d, Odds Ratio, Relative Risk

Select two binary (2-category) variables to build a 2x2 contingency table:
a = Exposed + Outcome  |  b = Exposed + No Outcome
c = Unexposed + Outcome  |  d = Unexposed + No Outcome
Results: OR, 95% CI of OR

🔬 Factor Analysis (EFA)

📖 เมื่อไหร่ควรใช้ Factor Analysis ?

วัตถุประสงค์: ลดจำนวนตัวแปร หาโครงสร้างปัจจัย (Factor) ที่ซ่อนอยู่

เงื่อนไข:

  • ตัวแปร Numeric อย่างน้อย 3 ตัว
  • KMO >= 0.5
  • Bartlett's test มีนัยสำคัญ

ข้อมูลที่ต้องมี: ตัวแปรเชิงปริมาณหลายตัว (เช่น ข้อคำถามแบบสอบถาม)

ตัวอย่าง: วิเคราะห์องค์ประกอบของแบบวัดความพึงพอใจ 20 ข้อ

📊 Cross Tabulation

📖 เมื่อไหร่ควรใช้ Cross Tabulation ?

วัตถุประสงค์: วิเคราะห์ความสัมพันธ์ตัวแปรเชิงกลุ่ม แสดง Observed, Expected, Row%, Col%, Residuals

เงื่อนไข: ตัวแปรเชิงกลุ่ม 2 ตัว

ตัวอย่าง: วิเคราะห์ เพศ x ระดับการศึกษา x ความพึงพอใจ

📐 Multiple Comparisons (Post-hoc)

📖 เมื่อไหร่ควรใช้ Post-hoc ?

วัตถุประสงค์: เปรียบเทียบรายคู่หลัง ANOVA พบความแตกต่าง

วิธีที่มี: Tukey HSD, Bonferroni, Scheffe

ข้อมูล: ตัวแปรตาม (DV) + ตัวแปรกลุ่ม (Factor) 3+ กลุ่ม

🔄 Bootstrap Confidence Interval

📖 เมื่อไหร่ควรใช้ Bootstrap CI ?

วัตถุประสงค์: ประมาณค่า CI โดยไม่ต้องสมมติการแจกแจง

ข้อดี: ใช้ได้กับข้อมูลทุกรูปแบบ ไม่ต้องปกติ

ตัวอย่าง: หา 95% CI ของค่าเฉลี่ย/ค่ามัธยฐาน เมื่อข้อมูลไม่ปกติ

📏 Z-Score & Percentile Rank

📖 เมื่อไหร่ควรใช้ Z-Score ?

วัตถุประสงค์: แปลงค่าข้อมูลเป็นคะแนนมาตรฐาน และหาตำแหน่งเปอร์เซ็นไทล์

ตัวอย่าง: เปรียบเทียบคะแนนสอบต่างวิชา, หาตำแหน่งสัมพัทธ์

📊 Multicollinearity Analysis (VIF)

📖 เมื่อไหร่ควรใช้ VIF ?

วัตถุประสงค์: ตรวจสอบ Multicollinearity ระหว่างตัวแปรอิสระก่อนทำ Regression

เกณฑ์: VIF < 5 = OK, VIF 5-10 = Moderate, VIF > 10 = Severe

ตัวอย่าง: ตรวจสอบก่อนวิเคราะห์ Multiple Regression

📊 Charts & Visualization

📖 Data Visualization

ประเภทกราฟ: Histogram, Boxplot, Q-Q Plot, Scatter Plot

📈 Partial Correlation

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: วิเคราะห์ความสัมพันธ์ระหว่าง 2 ตัวแปร โดยควบคุมอิทธิพลของตัวแปรที่ 3

ตัวอย่าง: ความสัมพันธ์ระหว่าง GPA กับ ชั่วโมงเรียน โดยควบคุม IQ

📈 Hierarchical Regression

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: วิเคราะห์การถดถอยแบบลำดับขั้น ดูว่าแต่ละ block เพิ่ม R² ได้เท่าไหร่

ตัวอย่าง: Block 1: เพศ อายุ → Block 2: แรงจูงใจ ทัศนคติ → พยากรณ์ผลสัมฤทธิ์

📈 ROC Curve / AUC

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: ประเมินความแม่นยำของ model ที่พยากรณ์ผลลัพธ์แบบ 2 ค่า

ตัวอย่าง: ประเมิน model พยากรณ์การเป็นโรค (ป่วย/ไม่ป่วย)

📊 ICC (Intraclass Correlation)

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: วัดความสอดคล้องระหว่างผู้ประเมิน/การวัดซ้ำ

ตัวอย่าง: ความสอดคล้องระหว่างผู้เชี่ยวชาญ 3 คน ในการให้คะแนน

📊 Split-Half Reliability

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: ทดสอบความเชื่อมั่นโดยแบ่งข้อคำถามเป็น 2 ส่วน

📐 McNemar Test

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: ทดสอบการเปลี่ยนแปลงของตัวแปร Binary ก่อน-หลัง

ตัวอย่าง: สอบผ่าน/ไม่ผ่าน ก่อน-หลังอบรม

📐 Fisher's Exact Test

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: ทดสอบความสัมพันธ์ตัวแปร 2x2 เมื่อ Expected < 5

หมายเหตุ: ใช้แทน Chi-Square เมื่อ cell มีค่าน้อย

📐 Cochran's Q Test

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: เปรียบเทียบสัดส่วน (Binary 0/1) จาก 3+ เงื่อนไข ในกลุ่มเดียวกัน

ตัวอย่าง: ผ่าน/ไม่ผ่าน 3 รอบการทดสอบ

🎯 Power Analysis (Sample Size)

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: คำนวณขนาดตัวอย่างที่เหมาะสม

ใช้ก่อน: เก็บข้อมูล เพื่อวางแผนจำนวนกลุ่มตัวอย่าง

Effect Size ตามเกณฑ์ Cohen:
t-test: Small=0.2, Medium=0.5, Large=0.8
ANOVA (f): Small=0.1, Medium=0.25, Large=0.4
Correlation: Small=0.1, Medium=0.3, Large=0.5
Chi-Square (w): Small=0.1, Medium=0.3, Large=0.5

📊 Welch ANOVA

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: เปรียบเทียบ 3+ กลุ่ม เมื่อความแปรปรวนไม่เท่ากัน

ใช้แทน: One-way ANOVA เมื่อ Levene's test มีนัยสำคัญ

📐 Dunnett's Test

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: เปรียบเทียบทุกกลุ่มกับกลุ่มควบคุม (Control)

📐 Median Test

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: เปรียบเทียบ Median ระหว่างกลุ่ม

ใช้แทน: Kruskal-Wallis เมื่อต้องการทดสอบ Median โดยเฉพาะ

📐 Runs Test

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: ทดสอบว่าข้อมูลเป็นแบบสุ่มหรือไม่ (Randomness)

📐 Kolmogorov-Smirnov 2-Sample

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: ทดสอบว่า 2 กลุ่มมาจากการแจกแจงเดียวกันหรือไม่

🔬 Cluster Analysis (K-Means)

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: จัดกลุ่มข้อมูลที่คล้ายกันเข้าด้วยกัน

ตัวอย่าง: แบ่งลูกค้าเป็น segment ตามพฤติกรรม

🔬 Discriminant Analysis

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: จำแนกกลุ่มจากตัวแปรอิสระ (2 กลุ่ม)

ตัวอย่าง: จำแนกนักเรียนสอบผ่าน/ไม่ผ่าน จากคะแนนย่อย

📋 Missing Value Analysis

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: ตรวจสอบข้อมูลที่หายไป (Missing Data) ก่อนวิเคราะห์

📈 Q-Q Plot

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: ตรวจสอบการแจกแจงปกติแบบกราฟ

อ่านผล: ถ้าจุดอยู่บนเส้นตรง = ข้อมูลปกติ

📈 Survival Analysis

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: วิเคราะห์ระยะเวลาจนเกิดเหตุการณ์ (time-to-event)

เครื่องมือ: Kaplan-Meier curve, Log-Rank test, Cox Regression

ตัวอย่าง: ระยะเวลารอดชีวิต, เวลาจนลาออก, เวลาจนเครื่องเสีย

ข้อมูล: Time (ระยะเวลา) + Event (0=censored, 1=เกิดเหตุ) + Group (ถ้าเปรียบเทียบ)

📈 Time Series Analysis

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: วิเคราะห์แนวโน้ม ฤดูกาล และพยากรณ์ข้อมูลอนุกรมเวลา

เครื่องมือ: Trend, Seasonal decomposition, Moving Average, Forecast

ตัวอย่าง: ยอมขายรายเดือน, จำนวนผู้ป่วยรายสัปดาห์

📊 One-Sample t-test

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: ทดสอบว่าค่าเฉลี่ยของกลุ่มตัวอย่างแตกต่างจากค่าที่กำหนด (Test Value) หรือไม่

เงื่อนไข:

  • ข้อมูลแจกแจงปกติ
  • ตัวแปรเชิงปริมาณ

ตัวอย่าง: ทดสอบว่าค่าเฉลี่ย IQ ของนักเรียนแตกต่างจาก 100 หรือไม่

📊 MANOVA (Multivariate ANOVA)

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: เปรียบเทียบค่าเฉลี่ยหลายตัวแปรตาม (DV) พร้อมกัน ระหว่างกลุ่ม

เงื่อนไข:

  • DV หลายตัว แจกแจงปกติแบบหลายตัวแปร
  • ความแปรปรวนร่วมเท่ากัน (Box's M)

ตัวอย่าง: เปรียบเทียบคะแนนวิชาคณิตฯ + วิทย์ + อังกฤษ ระหว่าง 3 โรงเรียน

📐 Sign Test

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: ทดสอบทิศทางความแตกต่างก่อน-หลัง (ดีขึ้น/แย่ลง) โดยไม่สนขนาดผลต่าง

เงื่อนไข:

  • ข้อมูลจับคู่ (Paired)
  • ไม่ต้องการสมมติฐานเรื่องการแจกแจง

ตัวอย่าง: ทดสอบว่าคะแนนหลังเรียนดีขึ้นจากก่อนเรียนหรือไม่

หมายเหตุ: เรียบง่ายกว่า Wilcoxon — ใช้เมื่อข้อมูลเป็น Ordinal หรือมี Outlier มาก

📐 Binomial Test

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: ทดสอบว่าสัดส่วนที่สังเกตได้แตกต่างจากสัดส่วนที่คาดหวังหรือไม่

เงื่อนไข:

  • ตัวแปรมี 2 ค่า (Binary: สำเร็จ/ไม่สำเร็จ, ใช่/ไม่ใช่)

ตัวอย่าง: ทดสอบว่าอัตราการสอบผ่าน 70% แตกต่างจาก 50% หรือไม่

🔄 Games-Howell Post-hoc Test

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: เปรียบเทียบรายคู่เมื่อความแปรปรวนไม่เท่ากัน

ใช้หลัง: Welch ANOVA หรือเมื่อ Levene's test มีนัยสำคัญ

ตัวอย่าง: เปรียบเทียบคะแนนรายคู่ 3 กลุ่ม ที่มี SD ต่างกันมาก

📈 Multiple Regression

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: พยากรณ์ตัวแปรตามจากตัวแปรอิสระหลายตัว

เงื่อนไข:

  • DV เป็นตัวเลขต่อเนื่อง
  • ความสัมพันธ์เชิงเส้น
  • ไม่มี Multicollinearity

ตัวอย่าง: พยากรณ์ GPA จากชั่วโมงอ่านหนังสือ + แรงจูงใจ + IQ

✅ Cohen's Kappa

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: วัดความสอดคล้องระหว่างผู้ประเมิน 2 คน (Inter-rater Agreement)

เงื่อนไข:

  • ตัวแปรเชิงกลุ่ม (Categorical)
  • ผู้ประเมิน 2 คน ประเมินชุดเดียวกัน

เกณฑ์แปลผล: <0.20 เห็นด้วยน้อย, 0.21-0.40 พอใช้, 0.41-0.60 ปานกลาง, 0.61-0.80 ดี, 0.81-1.00 ดีมาก

✅ Confirmatory Factor Analysis (CFA)

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: ยืนยันโครงสร้างองค์ประกอบของแบบวัดที่กำหนดไว้ล่วงหน้า

ใช้หลัง: EFA เพื่อยืนยันโมเดล หรือใช้ตรวจสอบความตรงเชิงโครงสร้าง

ดัชนีที่ดู: Chi-square, CFI (≥0.90), TLI (≥0.90), RMSEA (≤0.08), SRMR (≤0.08)

✅ Item Analysis (Difficulty & Discrimination)

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: วิเคราะห์คุณภาพข้อสอบ — ค่าความยาก (p) และค่าอำนาจจำแนก (r)

เกณฑ์:

  • ค่าความยาก (p): 0.20-0.80 ถือว่าเหมาะสม
  • ค่าอำนาจจำแนก (r): ≥0.20 ถือว่ายอมรับได้

ข้อมูล: คะแนนรายข้อ (0/1 หรือคะแนนย่อย) ของนักเรียนทุกคน

🎯 คำนวณขนาดตัวอย่าง (Sample Size Calculator)

📖 วิธีใช้งาน

วัตถุประสงค์: คำนวณจำนวนกลุ่มตัวอย่างที่เหมาะสมก่อนเก็บข้อมูล

สูตรที่รองรับ:

  • Taro Yamane — เมื่อทราบจำนวนประชากร
  • Cochran — เมื่อไม่ทราบจำนวนประชากร
  • Krejcie & Morgan — ตารางสำเร็จรูป
  • G*Power — ตาม Effect Size / Power ของสถิติที่ใช้
  • W.G. Cochran (Proportion) — สำหรับสัดส่วน

เลือกสูตรการคำนวณ

Effect Size ตามเกณฑ์ Cohen:
t-test (d): Small=0.2, Medium=0.5, Large=0.8
ANOVA (f): Small=0.10, Medium=0.25, Large=0.40
Correlation (r): Small=0.10, Medium=0.30, Large=0.50
Chi-Square (w): Small=0.10, Medium=0.30, Large=0.50
Regression (f2): Small=0.02, Medium=0.15, Large=0.35

🔍 Homogeneity of Variance Test

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: ทดสอบว่าความแปรปรวนของกลุ่มเท่ากันหรือไม่

วิธีทดสอบ: Levene's Test, Bartlett's Test

ใช้ก่อน: Independent t-test, ANOVA เพื่อเช็ค Assumption

🧠 Path Analysis

📖 เมื่อไหร่ควรใช้?

วัตถุประสงค์: วิเคราะห์อิทธิพลทางตรงและทางอ้อมระหว่างตัวแปร

เงื่อนไข:

  • มีกรอบแนวคิด/โมเดลที่กำหนดทิศทางอิทธิพล
  • ตัวแปรเชิงปริมาณ

ตัวอย่าง: แรงจูงใจ → พฤติกรรมการเรียน → ผลสัมฤทธิ์ (Direct + Indirect Effect)
